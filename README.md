# 🎓 OOP Quiz Management System

A Java console-based quiz application developed as my **Object-Oriented Programming (OOP) Final Project**. The application allows students to answer programming-related quiz questions, automatically evaluates their responses, and generates a final score with a grade.

---

## Features

- 👤 Student Registration (Name & Student ID)
- 📝 Multiple Choice Questions (MCQs)
- ✅ True/False Questions
- ✔️ Automatic Answer Validation
- 📊 Score Calculation
- 🎓 Automatic Grade Generation
- 📄 Result Card Display
- 💻 Console-Based Interface
